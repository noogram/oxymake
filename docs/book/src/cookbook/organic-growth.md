# Growing a Workflow Organically

> Coming soon.

This page will illustrate how real research workflows evolve over time,
covering:

- **Starting small**: a 3-rule exploratory workflow
- **Adding complexity incrementally**: new rules never invalidate existing
  cached results thanks to content-addressable caching
- **Workflow composition**: splitting large workflows across files with
  `include` directives
- **Snapshot milestones**: saving and comparing workflow states as research
  progresses
- **Run annotations**: using `ox run --note` to create a lightweight research
  lab notebook from the execution history
